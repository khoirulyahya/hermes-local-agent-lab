# Telegram Gateway Test Prompts

Use these prompts to test a Telegram gateway safely.

```text
cek sudah aktif?
provider dan model aktif apa?
cek MCP Trello sudah konek?
cek gateway status
baca list Doing di Trello, read-only
jangan ubah apa pun
```

For risky actions, require explicit approvals:

```text
approve card
approve plan
approve commit
approve push
```
