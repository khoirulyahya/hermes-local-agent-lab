# Voice Test Prompts

Use these as safe test cases for voice workflows.

```text
Kirim voice Bahasa Indonesia: "cek gateway aktif atau belum"
Kirim voice English: "check the Trello connection"
Kirim mixed language: "cek MCP Trello and summarize the card"
Minta balasan voice singkat jika TTS tersedia
```

Track each layer separately:

- voice received,
- voice transcribed,
- intent understood,
- response generated,
- voice output delivered.
